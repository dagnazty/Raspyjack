LINUX PRIVILEGE ESCALATION CHEATSHEET
======================================

ENUMERATION
  whoami && id                  Current user/groups
  uname -a                     Kernel version
  cat /etc/os-release           OS info
  cat /etc/passwd               User list
  cat /etc/shadow               Password hashes (if readable)
  cat /etc/sudoers              Sudo config
  sudo -l                      What can I sudo?
  find / -perm -4000 2>/dev/null   SUID binaries
  find / -perm -2000 2>/dev/null   SGID binaries
  find / -writable 2>/dev/null     Writable files
  getcap -r / 2>/dev/null          Capabilities
  crontab -l && ls /etc/cron*      Cron jobs
  ps aux                          Running processes
  netstat -tulnp                   Open ports
  env                             Environment variables
  cat /etc/fstab                  Mounted filesystems
  ls -la /tmp /var/tmp /dev/shm   World-writable dirs
